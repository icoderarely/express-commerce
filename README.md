# E-Commerce API

Small Node.js/Express API for an e-commerce project.

## Quick Start

- Install deps: `npm install`
- Run dev server: `npm run dev` or `nodemon index.js`

## Folders

- `routes/` API routes
- `models/` Mongoose models
- `middleware/` auth/role middleware
- `config/` app config (passport, etc)
- `upload/` uploaded files
